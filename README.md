# AIFX — The Synthetic Muse

A TypeScript + Next.js cinematic showcase that reinterprets a Renaissance-style scroll transition for AI film, commercial campaigns, music videos, trailers, and original IP.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Experience structure

- An original ET-Koo × KooLoo Renaissance-cinema tableau
- Scroll-scrubbed collapse into a persistent edition index
- WebGL particle, constellation, grid, and etched-image transition
- Five full-screen video chapters migrated from the source showcase
- Responsive menu, chapter navigation, sound controls, and reduced-motion support
